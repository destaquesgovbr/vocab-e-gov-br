# VCGE 2.0.3 - Vocabulário Controlado do Governo Eletrônico

!!! success "Versão Estável"
    Esta é a versão mais recente e estável do VCGE.

## Descrição

O **VCGE 2.0.3** é a versão atualizada do Vocabulário Controlado do Governo Eletrônico, utilizado para categorização de serviços e informações públicas.

## Namespace URI

```
http://vocab.e.gov.br/id/governo
```

## Autores

- [ePING](http://www.eping.e.gov.br/) - Padrões de Interoperabilidade de Governo Eletrônico

## Formatos Disponíveis

Os arquivos originais estavam disponíveis no portal do Governo Eletrônico:

| Formato | Descrição |
|---------|-----------|
| JSON | JavaScript Object Notation |
| SKOS (RDF/N3) | Simple Knowledge Organization System |
| CSV | Comma-separated values |
| SQL | Structured Query Language |
| PDF | Portable Document Format |

## Diferenças da versão 2011

A versão 2.0.3 trouxe:

- Atualização de termos obsoletos
- Novos termos para áreas emergentes
- Correções de inconsistências na hierarquia
- Melhoria nas notas explicativas
