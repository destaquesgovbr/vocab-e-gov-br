# EDGV 2.1 - Estruturação de Dados Geoespaciais Vetoriais

!!! success "Versão Estável"
    Especificação técnica para estruturação de dados geoespaciais vetoriais.

## Descrição

O **EDGV** (Estruturação de Dados Geoespaciais Vetoriais) é uma especificação técnica que define a estrutura de dados para informações geoespaciais vetoriais no Brasil.

## Namespace URI

```
http://vocab.e.gov.br/edgv/2.1
```

## Autores

- [CONCAR](http://www.concar.gov.br/) - Comissão Nacional de Cartografia do Governo Brasileiro

## Aplicação

O EDGV é utilizado para:

- Padronização de dados geoespaciais
- Intercâmbio de informações cartográficas
- Infraestrutura Nacional de Dados Espaciais (INDE)
