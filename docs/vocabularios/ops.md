# OPS - Ontologia de Participação Social

!!! warning "Em Desenvolvimento"
    Este vocabulário estava em desenvolvimento quando o site foi arquivado.

## Descrição

A **OPS** (Ontologia de Participação Social) é uma ontologia para modelar conceitos relacionados à participação social no governo.
