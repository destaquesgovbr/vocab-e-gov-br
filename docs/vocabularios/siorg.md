# SIORG - Estruturas Organizacionais

!!! warning "Em Desenvolvimento"
    Este vocabulário estava em desenvolvimento quando o site foi arquivado.

## Descrição

O vocabulário **SIORG** representa as estruturas organizacionais do Governo Federal brasileiro, baseado no Sistema de Informações Organizacionais do Governo Federal.

## Namespace URI

```
http://vocab.e.gov.br/siorg
```

## Autores

## Estrutura

O vocabulário SIORG modela:

- Órgãos da administração pública federal
- Hierarquia organizacional
- Unidades administrativas
- Relacionamentos entre órgãos
