# Sobre este Projeto

## O que é este site?

Este é um **espelho do antigo site vocab.e.gov.br**, que foi desativado há alguns anos. O conteúdo foi recuperado do [Internet Archive (Wayback Machine)](https://web.archive.org/web/20191222064047/http://vocab.e.gov.br/) e republicado para preservação histórica e referência.

## Por que preservar este conteúdo?

O **Repositório de Vocabulários e Ontologias do Governo Eletrônico** (e-VoG) foi uma iniciativa importante para:

- Padronização semântica de dados governamentais
- Interoperabilidade entre sistemas de governo
- Dados abertos e linked data no setor público brasileiro

Os vocabulários desenvolvidos, especialmente o **VCGE** (Vocabulário Controlado de Governo Eletrônico), foram amplamente utilizados para categorização de serviços públicos e continuam sendo referência para iniciativas de dados abertos.

## O que foi modernizado?

Este espelho não é uma cópia exata do site original. Algumas modernizações foram feitas:

| Aspecto | Original | Modernizado |
|---------|----------|-------------|
| Framework | HTML estático + jQuery | MkDocs Material |
| Navegador VCGE | jstree (jQuery) | Alpine.js |
| Hospedagem | Servidor gov.br | GitHub Pages |
| Responsividade | Limitada | Completa |

## Conteúdo preservado

- ✅ Todas as páginas HTML principais
- ✅ Documentação de vocabulários
- ✅ Arquivos de dados (JSON, RDF, OWL)
- ✅ Funcionalidade de navegação em árvore do VCGE
- ⚠️ Alguns recursos externos não disponíveis
